## Featured Projects

### [Tandoor — E-commerce Platform](https://aonetandoor-2.vercel.app/)
Production storefront built on a headless architecture. Sanity CMS handles all content
operations without a single database query on the frontend. Cloudinary serves
optimised images at the edge. The checkout flow is clean, fast, and complete.

`Next.js` `Sanity CMS` `Tailwind CSS` `Stripe` `Cloudinary` `Vercel`

---

### [ProctorShield — Real-time Monitoring System](https://secure-stream-ai-detection.vercel.app/)
A secure exam proctoring engine that detects facial landmarks in real time, monitors
live sessions over WebSockets, and flags violations with event-based precision.
Built to handle concurrent sessions without degrading — because in an exam context,
latency is cheating.

`Node.js` `Socket.io` `React` `MongoDB` `Express` `face-api.js` `MediaPipe` `WebRTC` `Cloudinary`

---

### [HireTrack Pro — Job Search Command Centre](https://hiring-dashboard-pro.vercel.app/)
A full personal CRM for job seekers — recruiter tracking, daily application counter,
streak system, interview pipeline, Draft Vault, Notes, Global Search, and analytics.
Everything persisted locally with zero backend dependency. Built because spreadsheets
are not a product.

`React` `Zustand` `Recharts` `Fuse.js` `Tailwind CSS` `IndexedDB` `react-hot-toast`
